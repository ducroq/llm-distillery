## Key Dimensions
1. Ecosystem Health (30%) - Species/biodiversity recovery
2. Pollution Reduction (25%) - Air/water quality improvements
3. Restoration Scale (15%) - Hectares protected/restored
4. Recovery Speed (10%) - How fast nature bounces back
5. Permanence (8%) - Protected status
6. Connectivity (5%) - Wildlife corridors
7. Historical Validation (4%) - Proven before (ozone, acid rain)
8. Attribution (3%) - Policy-driven recovery
